# Chat-Bot
Програмний модуль застосування штучних нейронних мереж для розвязання прикладних задач.
Як запустити
1. Створи базу MySQL з назвою `chatbot`.
2. Імпортуй файл `DB/chatbot.sql` у MySQL Workbench.
3. Створи `.env` файл за зразком `.env.example`.
4. Встанови залежності: pip install -r requirements.txt
5. Запусти застосунок: python main.py
